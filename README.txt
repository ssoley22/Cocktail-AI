==================================================
        PROJECTE: SMART COCKTAIL AI 🍹
==================================================

Aquest document explica com posar en marxa el projecte en un ordinador nou (Windows/Mac/Linux).

--- 1. REQUISITS PREVIS ---
1. Tenir instal·lat Python (3.10 o superior).
2. Tenir instal·lat Git.

--- 2. CONFIGURACIÓ INICIAL ---
1. Clonar el repositori:
   git clone https://github.com/ssoley22/Cocktail-AI.git

2. Crear l'entorn virtual (venv):
   python -m venv env

3. Activar l'entorn:
   - Windows: 
        Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
        .\env\Scripts\activate
   - Linux/Mac: source env/bin/activate

4. Instal·lar les llibreries:
   pip install -r requirements.txt
